# hackatweet

Bonjour!

Il s'agit d'un projet dont l'objectif est de créer un site similaire à Tweeter ( X ).

### Front End

La partie front end du projet a été codée en *Javascript* avec *React*, *Redux*, *Next* et *Ant Design*.

### Back End

Le partie back end a été développée avec l'infrastructure **Express** pour Node.js, **MongoDB** pour gérer la base de données, les modules **mongoose**, **dotenv**, **bcrypt**, **uid2** et **moment**.

Afin de tester le projet, il faudra installer les modules nécessaires et si vous le souhaitez, utiliser MongoDB pour la gestion des données des différents tickets de train.
